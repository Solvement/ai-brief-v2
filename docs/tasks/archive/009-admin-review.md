# Task 009: Admin review flow

## Goal

Create a simple review and publishing workflow.

## Acceptance Criteria

- Content status flow exists.
- Admin page shows draft / needs_review items.
- Admin can edit AI output fields.
- Admin can publish/archive content.
- Public pages only show published content.
- Quality warnings appear for low-confidence or incomplete items.

## Do Not Do

- Do not add complex permissions before MVP.
- Do not publish low-confidence content by default.
