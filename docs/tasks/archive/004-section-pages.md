# Task 004: Section pages

## Goal

Build News, Models, Tools, Playbooks, and Learn pages.

## Acceptance Criteria

- Each page exists.
- Each page lists relevant published content.
- Basic filters for tag, audience, and action label exist.
- Empty states exist.
- Layout is reusable.

## Do Not Do

- Do not build admin review flow.
- Do not add real external data sources.
