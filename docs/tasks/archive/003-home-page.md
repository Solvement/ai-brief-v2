# Task 003: Home page

## Goal

Build Home as a daily AI decision page.

## Acceptance Criteria

- Home includes:
  - Hero Brief
  - Must Read
  - Try Today
  - Model Radar
  - Playbook of the Day
  - Deep Read / Learn
- Uses seed data through query functions.
- No hard-coded data inside UI components.
- Cards link to detail pages.
- Empty states exist.
- Responsive layout works.

## Do Not Do

- Do not connect ingestion.
- Do not implement AI evaluation.
