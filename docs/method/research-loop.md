# 自进化研究循环 — 找问题 · 拆问题 · 科研不照抄

> Kevin 2026-06-09 定，写进 harness 复用。这是 L1 自进化的工作方式本身。
> 配套：[../../RULES.md](../../RULES.md) #18、[../agents/README.md](../agents/README.md)（跨模型对抗审）。

## 为什么有这份（核心原则）

我们做的是**自进化 agent**。网上方法更多，但**我们不照抄**——我们对**自己 deep dive 的材料做科研**，找**最适合我们自己**的结构优化。论文 / 项目 = **镜子和假设，不是配方**。

> L1 自进化 = agent 研究自己的处境、在自己语料上验证假设、进化自己的结构。
> 照抄 = 把 agentmemory 的栈搬过来。科研 = 在我们的语料 + 用例上验证"混合检索对我们到底有没有用、用哪种"，甚至自创一条 papers 都没有的路。

## 铁律

1. **读论文/项目要当镜子**：不只问"哪部分对我有用"，更要问"**我们的结构是不是有它正好治的病 / 它适不适合我们**"。
2. **自审找不出问题**——必须换方式找（下面 B）。
3. **不照抄**——多法竞赛、**指标当裁判**、选最适合我们的（接近就都做完比结果）。
4. **弱测试会骗人**——先把裁判（benchmark）做强，再比方法。

## 循环（可复用）

- **A. 批判性看两边**：论文/项目 + **我们自己的项目**。读论文当镜子照自己。
- **B. 找问题（自审找不出，换方式）**：① **真跑 / 真测**（live、对抗查询，不是"门过了就行"）② **跨模型对抗审**（codex ⇄ claude，generator≠critic **升到架构层**，不只内容层）③ 论文当镜子。
- **C. 拆**：大问题 → 小问题。
- **D. 科研不照抄**：每个小问题 → **多个假设方法**（含但不限于论文做法 + 我们自己的想法）→ **竞赛** → 指标判 → 选**最适合我们**的；分接近就都做完比结果。
- **E. 强化测试**：弱指标会骗（见下 retro）→ 做**对抗 / 更难的 benchmark** 当裁判，再比方法。
- **F. 迭代**：问题 + 决策记进 `task-board.md` / `memory/changelog.md`。

## Worked example（2026-06-09 这次的 retro）

**怎么发现的问题：**
- 你**指**（那张 325 节点毛球图）。
- 我**真跑** `scripts/kg/recall.mjs`——demo 暴露"查代码结构索引器→ agentmemory 排第 1、codegraph 第 2"，纯向量区分不开。
- 你**逼我把论文当镜子**（适不适合我们）→ 才发现**我们走的是纯向量 = agent-memory 领域淘汰的最弱路**（agentmemory 用三路混合+RRF、memoryagentbench 实测纯向量不够）。
- `/browse` 视觉审抓到 MemoryAgentBench 的 mermaid 报 Syntax error。

**为什么之前没发现（根因，要害）：**
1. **只自审**——build 过 + 门过 = 我以为没事。
2. **裁判太弱**——recall holdout 只有 5 条简单题，`recall@3=1.0` **骗了我**（弱测试会给假绿）。
3. **读论文找"对我有用"，不找"我哪里错"**——确认而非证伪。
4. **架构层 generator=critic**——我自己建、自己判架构，没有独立对手。
5. **结构门冒充质量门**——validate-mind-palace 只查"mermaid 有围栏""facet 非空"，不查"能编译""召回好不好"。

**这次怎么解：**
- 检索精度做**科研**：纯向量 vs 混合(BM25+向量+typed图)+RRF vs rerank vs 我们自创 → **扩充对抗 benchmark 当裁判** → 选最适合我们的。
- **codex 对抗审**整个 Mind Palace 架构（找 claude 自审漏的）。
- deep-dive facet **加一格**：「出发点是什么 / 这问题我们有没有 / 有哪几种解法 / 我们选哪个、为什么」——否则只是开拓思路，反哺不了结构决策。

**接下来（优先级）：**
1. 先攻**检索精度**（命门 + 已暴露弱 + 有镜子）。
2. 并行 **codex 架构对抗审**。
3. **冲突消解**（memoryagentbench 说的全场盲区 ≤6%）排进 L1 核心迭代——新论文推翻旧记忆怎么办，正是我们 `contradicts` 边该 handle 的。

## 反模式（这次踩的，以后别踩）

- 自审当质量门 · 弱 benchmark 当裁判 · 读论文只找有用的（不照镜子）· 单路走到死 · 结构门冒充质量门 · **照抄而非科研**。
